# 送付文
お疲れ様です！
ジムの閑散期対策を考えてみました。

## 概要
LINE上で「来店バッジ × 自動リマインド」を導入する

## 目的
冬場（11〜2月）の閑散期にお客様の来店を促進し、リピート率と稼働率の改善を図る。
「LINEでバッジが貯まる＋自動リマインドで来店を思い出す」ことで、再予約・再来店を促す。

## 具体内容
①来店バッジ機能（モチベーション向上）
LINE公式アカウントのメニューに「バッジ確認」ボタンを設置。
お客様が押すと、来店回数に応じて⭐が増える「ポイントカード風画像」が表示され流。
来店10回などの節目でプロテインなどの特典。

②自動リマインド機能（来店促進）
最終来店日から2週間経過したお客様に、自動で「次回予約をお忘れではありませんか？」というLINEメッセージを送信。
リマインドにはEPARK予約ページへのリンクを添付し、スムーズに予約へ誘導。
